# Telangana-Weather-Analysis-Dashboard-Using-Power-BI

## Project Overview

This project focuses on analyzing Telangana weather data using **Microsoft Power BI**. The dashboard provides insights into rainfall, temperature, humidity, and wind speed patterns across multiple districts and mandals of Telangana.

The objective of this project is to transform raw weather data into meaningful visual insights that can help understand climate trends, seasonal variations, and environmental conditions.

---

# Objectives

* Analyze weather trends across Telangana
* Monitor rainfall patterns over different years
* Compare temperature variations between districts
* Study humidity and wind speed behavior
* Build an interactive dashboard for data-driven insights

---

# Tools & Technologies Used

* **Power BI Desktop**
* **Power Query** (Data Cleaning & Transformation)
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel / CSV Files**

---

# Dataset Information

The dataset contains weather-related information collected from Telangana districts and mandals.

### Dataset Columns

* District
* Mandal
* Date
* Rain (mm)
* Min Temp (°C)
* Max Temp (°C)
* Min Humidity (%)
* Max Humidity (%)
* Min Wind Speed (Kmph)
* Max Wind Speed (Kmph)

---

# Data Cleaning & Transformation

Data preprocessing was performed using **Power Query**.

### Steps Performed

* Combined multiple files from yearly folders
* Removed duplicate records
* Handled null/missing values
* Changed incorrect data types
* Renamed columns for consistency
* Created Date hierarchy columns
* Extracted Year and Month information
